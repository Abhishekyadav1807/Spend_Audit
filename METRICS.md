# Metrics Plan (Draft)

## North Star Metric

Completed audits per week with valid recommendations shown to the user.

## Why This Metric

This is a lead-generation product where value is created at audit completion, not passive browsing.

## 3 Input Metrics

1. Landing page to audit-start conversion rate
2. Audit-start to audit-complete conversion rate
3. Audit-complete to lead-capture conversion rate

## First Instrumentation

- Track page view, audit start, audit submit, audit result rendered, lead form submit, and share link creation events.

## Pivot Trigger

If audit completion stays below 20% of starts after UX iteration and interview feedback, revisit form complexity and positioning.
